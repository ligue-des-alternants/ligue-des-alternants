# @ligue-des-alternants/typescript-config

Shared TypeScript configurations for the Ligue des Alternants monorepo.

## Usage

### Base Config

```json
{
  "extends": "@ligue-des-alternants/typescript-config/base"
}
```

### React Config

```json
{
  "extends": "@ligue-des-alternants/typescript-config/react"
}
```

### Node Config

```json
{
  "extends": "@ligue-des-alternants/typescript-config/node"
}
```
