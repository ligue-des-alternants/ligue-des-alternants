# @ligue-des-alternants/eslint-config

Shared ESLint configurations for the Ligue des Alternants monorepo.

## Usage

### Base Config

```js
import config from '@ligue-des-alternants/eslint-config/base';

export default config;
```

### React Config (with Astro support)

```js
import config from '@ligue-des-alternants/eslint-config/react';

export default config;
```

### Node Config (for Strapi)

```js
import config from '@ligue-des-alternants/eslint-config/node';

export default config;
```
