# @ligue-des-alternants/prettier-config

Shared Prettier configuration for the Ligue des Alternants monorepo.

## Usage

In your app's `package.json`:

```json
{
  "prettier": "@ligue-des-alternants/prettier-config"
}
```

Or in `.prettierrc.json`:

```json
"@ligue-des-alternants/prettier-config"
```
